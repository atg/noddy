Alert.show(title, message, buttons)

Shows an alert.

  • title String – the title of your alert.
  • message String – the body of your alert message.
  • buttons Array – an array of buttons to display.

Example

Alert.show("My Title", "My awesome body!", ["Yes", "No"]);
